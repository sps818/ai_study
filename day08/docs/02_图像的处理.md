### 图像的读取：
- 1. Matplotlib
  - 数据科学三剑客
  - 数据可视化
  - 高仿Matlab
    - plt.imread()
    - plt.imshow()
    - plt.imsave()
  - 没有内置图像处理的功能
  - 需要手动去操控NumPy数组

- 2. OpenCV
  - 平面图像处理金标准！
  - C++
  - 使用 Python 调用
  - pip installl opencv-python

- 3. PIL: Python Imaging Library
  - Python 内置库
  - PyTorch 无缝衔接
